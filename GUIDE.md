## 기본 사용법

### 1. 덱 만들기
1. 홈 화면에서 우측 하단 **+ 버튼** 클릭
2. 덱 이름 입력 후 **만들기**

### 2. 카드 추가

#### 빈칸 타입 (핵심 기능)
1. 덱 진입 → **카드 추가** → **빈칸** 탭
2. 빈칸으로 만들 부분을 `{{단어}}`로 감싸서 입력
3. **한 줄 = 한 장의 카드**

**입력 예시:**
```
대한민국의 수도는 {{서울}}이다
{{산소}}의 원소기호는 {{O}}이다
The past tense of "go" is {{went}}
```

- 한 줄에 여러 개의 `{{}}`도 가능
- 입력 중 아래에 실시간 미리보기가 표시됨

#### 기본 타입
1. **기본** 탭 선택
2. 앞면(질문)과 뒷면(답)을 각각 입력
3. **카드 추가** 클릭

#### 파일 가져오기
1. **가져오기** 탭 선택
2. CSV, TSV, 또는 TXT 파일 선택
3. 미리보기 확인 후 **가져오기** 클릭

지원 형식:
| 파일 형식 | 구분자 | 예시 |
|-----------|--------|------|
| CSV | 쉼표(,) | `질문,답` |
| TSV | 탭 | `질문	답` |
| TXT (빈칸) | - | `수도는 {{서울}}이다` |

### 3. 카드 편집 / 삭제
- 카드 목록에서 카드 텍스트 부분을 클릭 → 편집 모달
- 내용 수정 후 **저장** 또는 **삭제**

## 학습하기

### 학습 시작
1. 덱 화면에서 **학습 시작** 클릭
2. 학습 설정 모달에서 옵션 선택:

| 옵션 | 설명 |
|------|------|
| 학습 대기 카드 | 복습 시간이 된 카드만 (기본값) |
| 전체 카드 | 덱의 모든 카드 |
| 별표 카드만 | ★ 표시한 카드만 |
| 틀린 카드 (Box 1) | 새 카드 또는 틀려서 리셋된 카드 |
| 타이핑 모드 | 빈칸 답을 직접 키보드로 입력 |

3. **시작** 클릭

### 일반 모드
1. 빈칸이 `____`로 표시된 카드가 나옴
2. 카드를 **탭(클릭)** → 정답 표시
3. **알고 있어요** 또는 **모르겠어요** 선택

### 타이핑 모드
1. 빈칸 위치에 입력창이 나옴
2. 답을 직접 입력 후 **확인** (또는 Enter)
3. 초록색 = 정답, 빨간색 = 오답 (정답이 함께 표시됨)
4. **알고 있어요** 또는 **모르겠어요** 선택

### 키보드 단축키

| 키 | 동작 |
|----|------|
| `Space` | 정답 확인 (일반 모드) |
| `Enter` | 정답 확인 (타이핑 모드) |
| `→` 또는 `2` | 알고 있어요 |
| `←` 또는 `1` | 모르겠어요 |
| `Ctrl+Z` / `Cmd+Z` | 되돌리기 |

## 간격 반복 시스템 (Leitner Box)

카드는 Box 1~5 단계로 관리됩니다.

```
맞으면 → Box 올라감 (복습 간격 증가)
틀리면 → Box 1로 리셋 (즉시 다시 복습)
```

| Box | 상태 | 다음 복습까지 |
|-----|------|--------------|
| Box 1 | 새 카드 / 틀린 카드 | 바로 |
| Box 2 | 1회 정답 | 1일 후 |
| Box 3 | 2회 연속 정답 | 3일 후 |
| Box 4 | 3회 연속 정답 | 7일 후 |
| Box 5 | 마스터 | 14일 후 |

---

## 별표 (즐겨찾기)

어렵거나 중요한 카드에 별표를 달아 따로 관리할 수 있습니다.

- **카드 목록**에서 ☆ 아이콘 클릭 → 별표 토글
- **학습 중**에도 카드 우측 상단 ☆로 별표 가능
- 학습 시작 시 **별표 카드만** 필터로 집중 복습

## 통계

홈 화면 > **통계** 버튼

| 항목 | 설명 |
|------|------|
| 🔥 연속 학습일 | 매일 공부하면 증가, 하루 빠지면 리셋 |
| 총 학습 카드 | 지금까지 학습한 카드 총 수 |
| 정답률 | 전체 정답 비율 |
| 학습한 날 | 실제로 학습한 일수 |
| 최장 연속 | 가장 길었던 연속 학습일 |
| 활동 캘린더 | 최근 30일간 학습량을 색상으로 표시 |
| 일별 기록 | 날짜별 정답/오답 수 |

## 덱 관리

카드 목록 우측 점 세개(⋮) 메뉴:

| 메뉴 | 설명 |
|------|------|
| 이름 변경 | 덱 이름 수정 |
| 학습 초기화 | 모든 카드를 Box 1로 리셋 (카드는 유지) |
| 삭제 | 덱과 모든 카드 삭제 |

## 데이터 관리

### 내보내기
- **덱별 내보내기**: 덱 상세 화면 > 헤더의 다운로드(↓) 아이콘 → `.txt` 파일
- **전체 백업**: 홈 화면 > **전체 백업** 버튼 → `.json` 파일

### 데이터 저장 위치
모든 데이터는 프로젝트 폴더의 `data.json` 파일에 저장됩니다.
브라우저를 바꾸거나 캐시를 지워도 데이터가 유지됩니다.
